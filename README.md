# streaming-site-design-logbook
该项目用于记录在线影视网站的页面设计特征，对首页布局、栏目逻辑与更新节奏进行归档说明。通过持续整理公开页面信息建立结构化资料库，用于观察不同影视平台在信息呈现方面的差异。仓库仅保存公开资料，不包含视频文件或播放链接。
